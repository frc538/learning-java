# Lesson 02 - Numeric Operations

## Topics

This lesson introduces:

- Integers
- Arithmetic Operators
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Modulus Division
- Concatenation with Arithmetic
- Floating Point Numbers
- The Java Math Library
  - Constants
  - sqrt
  - abs
  - pow
  - sin, cos, tan
  - asin, acos, atan
  - ceil, floor, round
  - exp, log, log10
  - max and min
  - hypot
  - atan2
